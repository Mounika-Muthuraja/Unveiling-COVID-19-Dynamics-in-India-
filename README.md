# Unveiling COVID-19 Dynamics in India
### End-to-End Data Analytics & Automation Project

## Project Overview

This project analyzes the spread and impact of COVID-19 across India using real pandemic datasets. It demonstrates a complete data analytics pipeline including data extraction, cleaning, transformation, visualization, and automation.

The goal of this project is to transform raw COVID-19 datasets into meaningful insights and automated reports using modern data analytics tools.

## Project Workflow

Raw COVID Data  
↓  
SQL ETL Processing  
↓  
Excel Analysis  
↓  
Power BI Visualization  
↓  
n8n Automation + AI Reports

## Key Insights

- India recorded **32M+ confirmed COVID cases**
- **Recovery rate exceeded 97%**
- **524M+ COVID tests conducted**
- Vaccination rollout significantly reduced severe cases
- Some states showed higher positivity rates indicating possible under-testing
