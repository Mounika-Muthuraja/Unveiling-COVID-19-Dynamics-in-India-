# 🤖 AI-Powered COVID Situation Report Automation (n8n + GenAI)

## Project Overview

This project builds an automated pipeline that generates **daily COVID-19 situation reports for India** using workflow automation and Generative AI.

The system processes COVID data, detects high-risk states using SQL analytics, generates an AI-based summary, and automatically sends a professional email report to stakeholders.

The goal is to transform raw public health data into **clear, actionable intelligence** using automation and AI.

---

# 🎯 Problem Statement

Public health data is often available in raw numerical tables. However, decision-makers require:

* Aggregated data
* Trend interpretation
* Risk detection
* Simplified communication
* Automated report delivery

Manually analyzing and summarizing this data daily is time-consuming and inefficient.

This project solves the problem by building an **automated reporting pipeline** that converts structured data into a simplified daily health briefing using AI.

---

# 🔄 n8n Workflow Steps

The workflow runs automatically on a schedule and performs the following steps:

### 1️⃣ Data Extraction

A PostgreSQL node retrieves processed COVID data from the analytics database.

### 2️⃣ Risk Detection

SQL logic identifies high-risk states using trend analysis.

Risk indicators include:

* **Three consecutive days of rising cases**
* **High test positivity rate (>10%)**
* **Vaccination growth slowing while cases rise**

### 3️⃣ Data Aggregation

A JavaScript **Code Node** processes all risky states and combines them into a single structured report.

Example processing logic:

* Calculate cases per million
* Format state statistics
* Combine risk indicators into a readable summary

### 4️⃣ AI Report Generation

The aggregated data is sent to a **Generative AI model** which produces a simplified narrative report.

The AI is instructed to:

* Explain trends in simple language
* Highlight high-risk states
* Summarize the national situation

### 5️⃣ Email Delivery

The final report is sent automatically using the **Gmail node** in HTML format.

Recipients receive a **daily COVID situation briefing** without manual intervention.

# 🧠 Risk Detection Logic (SQL)

High-risk states are detected using SQL window functions.

Example logic:

```sql
CASE 
WHEN daily_new_cases > cases_1_day_ago
AND cases_1_day_ago > cases_2_days_ago
AND cases_2_days_ago > cases_3_days_ago
THEN TRUE
END AS risk_consecutive_rise
```

Additional conditions:

* **High Positivity Rate** → `positive_test_rate > 10`
* **Vaccination Stall** → low vaccination growth + rising cases

These conditions help identify potential outbreak signals.

# 🧮 Data Aggregation (Code Node)

The n8n Code Node combines multiple state records into one AI input.

Example logic:

```javascript
const items = $input.all();
let reportText = "";

items.forEach((item, index) => {
  const s = item.json;
  reportText += `State: ${s.state}\n`;
  reportText += `New Cases: ${s.daily_new_cases}\n`;
  reportText += `Positivity Rate: ${s.positive_test_rate}%\n`;
  reportText += `Vaccination Rate: ${s.vaccination_rate}%\n\n`;
});
```

This prevents sending multiple requests to the AI model and creates a **single consolidated report**.
